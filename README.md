# CSV Minuten-Konverter

Konvertiert medizinische Monitor-CSV Dateien von Sekundendaten zu Minutenmittelwerten.

## Download

1. Oben auf **Actions** klicken
2. Den letzten erfolgreichen Build öffnen
3. Unter **Artifacts** herunterladen:
   - `CSV_Konverter_macOS` → für Mac
   - `CSV_Konverter_Windows` → für Windows

## Verwendung

Einfach die App öffnen – keine Installation nötig.

1. Eingabe-CSV wählen
2. Speicherort wählen
3. Nachkommastellen einstellen
4. „Konvertieren" klicken
